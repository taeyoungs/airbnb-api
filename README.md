# Airbnb API

REST API of the Airbnb Clone using Django REST Framework

## Stack

1. Django
2. Django REST Framework

## API Actions

- [x] JWT Authentication
- [x] Login (JWT)
- [x] Create Account
- [x] Add/Remove Room From Favourites
- [x] See Profile
- [x] List Rooms
- [x] Edit Profile
- [x] See Room
- [x] Create, Delete Room
- [x] Search Rooms
- [x] Search again map
